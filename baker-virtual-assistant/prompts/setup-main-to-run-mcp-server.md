# Prompt: Set up a new MCP Server

## Tasks

1. Using FastMCP library set up a new MCP Server.
2. Add `main.py` that will be used to initialize and run the MCP Server. Name the MCP Server "baker-virtual-assistant"
3. Add bash script `mcp-install.sh` that Install an MCP server in the Claude desktop app. 
4. Add bash script `mcp-run-dev.sh` that allows to Run MCP Server in dev mode. 

## Acceptance Criteria
In the end, the goal is to have an MCP Server up and running.
I should be able to run the server with MCP Inspector. Also, I should have the bash scripts for convinience to install and run the MCP Server in dev mode.
